A Note on <br/> Tags

To encourage correct usage, and provide greater control over their behaviour these tags have no behaviour by default, and not more than one will take effect if used multiple times. To use a break tag, apply one of the block helper classes. (eg: .block-until-alpha). 
        
            
br {
    display: none; // By default, breaks will do nothing unless paired with a .block-___-___ helper class - which ensures content flow isn't hampered where the break is uncessesary.
}

br + br {
    display: none !important; // Prevent misuse of break tags for mutli-line separation.
}
